HTML

HTML, son las siglás de Hyper Text Markup Language, que puede traducirse como lenguaje de marcas o marcado de hipertexto.

El lenguaje HTML, se emplea para crear las páginas web. Es muy fácil ver el código HTML de una página web, la opción exacta cambia de un navegador a otro y también puede cambiar de una versión a otra de un mimso navegador, pero suelen tener un nombre similar.

HTML se compone de etiquetas que se escriben entre los simbolos menor que y mayor que.

Historia de HTML

Los inicios del lenguaje HTML se remontan al año 1990, cunaod Tim Berners-Lee creo la primera página web.

Versiones de HTML

Tim Berners-Lee definió la primera versión de HTML en el año 1991.

En la actualidad, la última versión de HTML es HTML5.
